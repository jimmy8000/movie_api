Movies and Users API Documentation

This documentation provides details on the available endpoints and their corresponding operations for the Movies and Users API.

Movie Operations

Request URL Response
GET /api/movies Retrieve a list of all movies
GET /api/movies/{movie_id} Retrieve details of a specific movie
POST /api/movies Create a new movie
PUT /api/movies/{movie_id} Update details of a specific movie
DELETE /api/movies/{movie_id} Delete a specific movie

User Operations

Request URL Response
GET /api/users Retrieve a list of all users
GET /api/users/{user_id} Retrieve details of a specific user
POST /api/users Create a new user
PUT /api/users/{user_id} Update details of a specific user
DELETE /api/users/{user_id} Delete a specific user
