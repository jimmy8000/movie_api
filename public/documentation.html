Movies and Users API Documentation

This documentation provides details on the available endpoints and their corresponding operations for the Movies and Users API.

Movie Operations

Request URL Body Response
GET /movies Retrieve a list of all movies
GET /movies/{movie_id} Retrieve details of a specific movie
POST /movies JSON object with movie data 
{
  "Title": "Example Movie",
  "Description": "A thrilling movie",
  "Genre": {
    "Name": "Thriller",
    "Description": "Exciting and suspenseful"
  },
  "Director": {
    "Name": "John Doe",
    "Bio": "A talented director"
  },
  "Actors": ["Actor1", "Actor2"],
  "ImagePath": "/path/to/image",
  "Featured": true
}
Status Code: `201 Created`
JSON object with the created movie 
{
  "_id": "uniqueMovieId",
  "Title": "Example Movie",
  "Description": "A thrilling movie",
  "Genre": {
    "Name": "Thriller",
    "Description": "Exciting and suspenseful"
  },
  "Director": {
    "Name": "John Doe",
    "Bio": "A talented director"
  },
  "Actors": ["Actor1", "Actor2"],
  "ImagePath": "/path/to/image",
  "Featured": true
}
GET /movies/genre/{Genre} Get description by genre
GET /movies/director/{Director} Get director data by director name
DELETE /users/{Username}/movies/{MovieID} Remove a movie from a user's favorites

User Operations

Request URL Body Response
GET /users Retrieve a list of all users
GET /users/{user_id} Retrieve details of a specific user
POST /users JSON object with user data 
{
  "Username": "exampleUser",
  "Password": "examplePassword",
  "Email": "user@example.com",
  "Birthday": "2000-01-01"
}
Status Code: `201 Created`
JSON object with the created user 
{
  "_id": "uniqueUserId",
  "Username": "exampleUser",
  "Password": "examplePassword",
  "Email": "user@example.com",
  "Birthday": "2000-01-01",
  "FavoriteMovies": []
}
PUT /users/{Username} JSON object with updated user data 
{
  "Username": "newUsername",
  "Password": "newPassword",
  "Email": "newEmail@example.com",
  "Birthday": "1990-01-01"
}
JSON object with the updated user 
{
  "_id": "uniqueUserId",
  "Username": "newUsername",
  "Password": "newPassword",
  "Email": "newEmail@example.com",
  "Birthday": "1990-01-01",
  "FavoriteMovies": []
}
DELETE /users/{Username} Delete a specific user
